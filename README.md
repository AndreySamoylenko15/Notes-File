# 📝 Notes Manager (Python)

Простий консольний застосунок для збереження нотаток у файл.

## 🚀 Можливості

Програма дозволяє:

* ➕ Додавати нові нотатки
* 📄 Переглядати всі нотатки
* 🗑 Очищати список нотаток
* 💾 Зберігати дані у файл (`notes.txt`)

---

## 🛠 Як запустити

```bash
python notes_manager.py
```

---

## 📌 Приклад роботи

```
1 - Add note
2 - Show notes
3 - Clear notes
0 - Exit

Choose option: 1
Enter note: Buy milk

Choose option: 2
Buy milk
```

---

## 📂 Як працює

* Всі нотатки зберігаються у файлі `notes.txt`
* Кожна нотатка записується з нового рядка
* Дані зберігаються між запусками програми

---

## ⚠️ Обмеження

* Немає редагування окремих нотаток
* Немає видалення конкретної нотатки
* Всі дані зберігаються в одному файлі
* Без захисту від помилок введення

---

## 📚 Навчальна мета

Проєкт створений для практики:

* роботи з файлами (`open`, `read`, `write`)
* обробки помилок (`try/except`)
* циклів і умов
* створення простого CLI
